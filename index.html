Actividad1 Trimestre 1

El objetivo de este ejercicio es crear tres interfaces web para una tienda online utilizando HTML5 y CSS3. Cada interfaz constará de una página de portada, una página de ejemplo de producto y una página de formulario de contacto. Debes mantener la estética consistente en todas las páginas para garantizar una experiencia de usuario coherente. La temática de la tienda la decide el alumno.

Requisitos: Portada (index.html): Diseña una página de inicio atractiva para la tienda en línea. Incluye un encabezado con el nombre de la tienda y una barra de navegación que enlace a las otras dos páginas (Ejemplo de producto y Formulario de contacto). Agrega elementos visuales, como imágenes relacionadas con la tienda y un breve texto introductorio. Ejemplo de Producto (producto.html): Crea una página que muestre un ejemplo de producto que se vende en la tienda en línea. Muestra una imagen del producto, su nombre, precio y una breve descripción. Añade un botón de "Agregar al carrito" o un enlace a la página de compra. Formulario de Contacto (contacto.html): Diseña una página que incluya un formulario de contacto para los clientes. El formulario debe solicitar al menos el nombre, dirección de correo electrónico y un mensaje. Añade un botón de "Enviar" para enviar el formulario.

Pautas de Diseño: Utiliza HTML5 y CSS3 para estructurar y diseñar las páginas web. Mantén una estética coherente en todas las páginas, utilizando colores, fuentes y estilos similares. Asegúrate de que las páginas sean visualmente atractivas y fáciles de navegar. Puedes elegir todos los elementos HTML y CSS que se quieran. Se valora el buen uso de las etiquetas y del diseño CSS.
